## misc things to later add to main description file

* pid - Kickstarter unique project ID [automatic]
* name - project title
* blurb - short description displayed on project list
* goal - amount of money to collect for the project to be successful
* pledged - total amount of money pledged
* state - project current status [canceled / failed / live / successful / suspended]
* country - country in which project is being developed
* currency - currency in which goal is set and pledges are collected
* deadline - end date of pledging
* created_at - date at which project page was created
* launched_at - date at which project page was made public and pledging started
* backers_count - total amount of backers
* creator_id - Kickstarter unique user ID of the creator [automatic]
* creator_name - Name / nickname of the project creator [Kickstater User]
* has_video - if the front page has at least one video embedded [TRUE / FALSE]
* location_id - Kickstarter unique city ID in which project is being developed [automatic]
* location_name - City in which project is being developed [possibly merged with location_id ?]
* comments_count - total amount of backer comments on project page [only backers can post comments]
* updates_count - total amount of updates made to the project page by the creator
* reward_value - minimal amount of money needed to unlock given reward [backers may choose to pay more than value!]
* reward_desc - reward description
* reward_backers_count - total amount of backers who paid for that reward
