The Wingman Formation and Pattern User Interface

This is a unique feature to Bionite:Â OriginsÂ Multiplayer and Single Player. The pilot will be able to control his/her wingmen/wingwomen in the game by using a unique HUD element called the "Formation and Pattern Interface". The wingmen are designed to 'Fly in Formation' or 'Attack in Patterns' based on the pilots commands. These formations and patterns can be set up prior to entering the match, then executed in game on the battle field by selecting the Formation or Pattern Number in the Interface Panel (located in the HUD). We are currently in the R&D phase of adding this feature to the App. In addition to preset 'plays' the Formation and Pattern interface has a 'custom' section. This section of the UI allows the pilot to create thier own 'plays' for the wingmen to follow using the familiar 'X' and 'O' system used in football/soccer.

Risks and challenges Learn about accountability on Kickstarter

This Game has been in development for over 2 years now. We have released more than a dozen alpha versions of the Game during its development. We are using a very stable and tested game engine, the Unreal Engine 3. The real challenges for us as developers is to make a game that will run smoothly on older machines. We have tested the game on new computers and laptops and the Game runs dead smooth. We plan on making the Game available to run on older machines, but it will take our best ideas and talent to optimize the Game to run smoothly on machines that are older than 7 years. The Video above is an exact snapshot of where we are in the development of the Game. The Beta is almost ready for testing and we plan on releasing the first closed Beta by the end of this year.

Theres is very little left to do to complete the final version of the Game. The initial single player version of the Game will not have complex cinematics, which are the bulk of work involved with a single player game. The single player version of the Game will have all the core elements from the Multiplayer version put into a scripted environment, pitting the player against the computer.

The delay in the release of our closed Beta for the Multiplayer version is due to the amount of content we need to texture to complete the Venus Map. We have retained two great texture artists to surface the roughly 300 models we have, but its taking more time than I originally estimated. We are currently about half way through texturing the models, but it will take 3 or 4 more months to finish them.

In the Pitch video, we have a great example of the quality of work being done. The Minotaur Gun Tower was recently textured by our new staff texture artist: Jayden.
